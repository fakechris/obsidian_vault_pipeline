# Reasoning模块是Agent的核心智能所在

> **一句话定义**: Reasoning模块负责理解意图、知识检索、推理决策和计划生成，模拟人类的思考过程

## 📝 详细解释

### 是什么？
推理模块涉及知识表示、逻辑推理、规划等核心能力。该模块接收感知模块的结构化输入，进行理解和分析，制定行动计划。推理质量直接决定了Agent能否正确完成目标任务

### 为什么重要？
推理能力是AI Agent智能水平的关键体现，是区分Agent能力强弱的核心因素
