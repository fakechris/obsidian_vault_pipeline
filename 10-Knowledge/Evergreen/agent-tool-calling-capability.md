# Agent通过工具调用扩展行动能力

> **一句话定义**: Agent的行动模块可以将决策转化为实际行动，通过调用外部工具、API和系统操作来完成任务

## 📝 详细解释

### 是什么？
Agent的行动能力不仅限于生成回复，还包括：1)信息检索（搜索、查询数据库）2)内容生成（文本、代码、图片）3)工具调用（调用外部API、执行函数）4)系统操作（修改数据、触发流程）。工具调用使Agent能够与外部系统集成，突破了传统AI仅限于输出的限制，是Agent执行复杂任务的关键能力

### 为什么重要？
工具调用能力是Agent实现实际价值的基础，使Agent能够操作真实世界的数据和系统，完成了从『知道』到『做到』的关键跨越
